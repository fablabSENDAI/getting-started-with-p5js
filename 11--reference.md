# Reference : 参考

- [**p5js公式チュートリアル**](https://p5js.org/tutorials/)
    - p5jsが提供しているチュートリアル集。まずは、ここを見てみるのをおススメします。

- [**p5js公式リファレンス**](https://p5js.org/reference/)
    - p5jsで使用できる関数を一覧できるページ。初級者から上級者まで、p5jsユーザーは頻繁に見ることになるページ

- [**Coding Train**](https://www.youtube.com/@TheCodingTrain)
    - processing / p5js界隈の大御所、ダニエル・シフマン先生のYouTubeチャンネル。多種多様なチュートリアル動画が頻繁にアップされる。

- [**Nature of Code**](https://natureofcode.com/)
    - シフマン先生が公開しているプログラムによって自然現象を模倣する方法を探す、チュートリアルサイト。中級者以上向けだけど、ここを一通りこなすとグッとプログラム筋が鍛え上げられる。（ちなみに[書籍版](https://www.yodobashi.com/product/100000009002173354/)もある。）

- [**Open Processing**](https://openprocessing.org/)
    - p5jsで作成されたコードを投稿して、共有できるサイト。デイリーコーディング愛好家たちに利用されている。

- [**Takawo**](https://openprocessing.org/user/6533/#sketches)
    - 国内デイリーコーディングの第一人者 takawo氏のアカウント。毎日少しでも積み重ねていく大切さと、自由気ままにプログラミングする楽しさに溢れている。


- **Ash Thorp**
    - モーショングラフィック、映画中での疑似インターフェイズデザイン棟を手掛けるデザイナー。p5jsの基礎が分かった後に見ると沁みる彼の作品を二つ以下に紹介しておく。
    - [MODULAR](https://www.youtube.com/watch?v=J8sCIG83y5s)
    - [FITC](https://www.youtube.com/watch?v=KzY9N5dPZug)
